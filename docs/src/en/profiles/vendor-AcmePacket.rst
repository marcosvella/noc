==========
AcmePacket
==========

.. toctree::
    :titlesonly:
    :glob:

    /profiles/AcmePacket.*

`AddPac.*` family of profiles support various
`AcmePacket Technology Co <http://www.acmepacket.com/>`_
network equipment. On February 4, 2013, Oracle announced that it has agreed
to acquire Acme Packet, the global provider of session border control technology
for service providers and enterprises.
