# Global configuration

Global settings applicable to all services


loglevel
~~~~~~~~

==================  ===================
**YAML Path**       loglevel
**Key-Value Path**  loglevel
**Environment**     NOC_LOGLEVEL
**Default Value**   info
==================  ===================

Possible values:

* critical
* error
* warning
* info
* debug
