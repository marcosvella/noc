

========================
managedobject DataStream
========================

.. contents:: On this page
    :local:
    :backlinks: none
    :depth: 1
    :class: singlecol

managedobject DataStream contains summarized :ref:`Managed Object<reference-managed-object>`
state, including capabilities, interfaces and topology

Filters

Access
